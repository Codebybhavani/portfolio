# Personal Portfolio Website

A responsive personal portfolio website showcasing my skills, education, projects, experience, achievements, and contact information as an aspiring Software Development Engineer.

## Features

- Responsive design
- About Me section
- Education details
- Technical skills
- Project showcase
- Internship experience
- Achievements and certifications
- Contact information and social links

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Projects Highlighted

- CodePen Clone
- Campus Nexus Platform
- Cosmetic Product Analysis Web Application
- Doctor Assistant Web Application

## Installation

```bash
git clone <repository-url>
cd Portfolio
```

Open `index.html` in your browser or use Live Server.
